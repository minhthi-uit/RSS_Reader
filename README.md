# RSS_Reader

## About Project
This application allows user to input a RSS url (eg: vnexpress/tinhte,...) and read the content.

### Outcome:
1. Can save to read later 
2. Pretty UI
3. SSO login

### Built with
- Language: Java
- IDE: Android Studio
- Firebase Authentication
